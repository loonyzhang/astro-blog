VS Code 的代码提示突然不好用了，不知道什么原因导致的，只好重装试试。

但是直接卸载重装相关配置还是存在的，需要卸载后删除 VS Code 以下相关目录（Windows 系统）

- `%APPDATA%\Code`
- `%USERPROFILE%\.vscode`
然后，在重新安装。
